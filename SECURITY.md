# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this action, please report
it privately to **security@wiz.io**. Do not open a public GitHub issue.

Please include:

- A description of the issue and its impact
- Steps to reproduce (proof-of-concept if available)
- Affected versions

We will acknowledge receipt within a few business days and keep you informed of the
resolution timeline.
